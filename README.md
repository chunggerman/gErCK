# gErCK
gErCK AI Platform

This repository contains the first working vertical slice of the gErCK AI Platform.  
It implements one complete capability: Ask for Knowledge, built using modular, test‑driven, and configuration‑driven patterns.
Purpose
This vertical slice is the foundation of the full gErCK AI Platform.  
It is intentionally small and simple so it can be built by a single person, while still following the architecture of the full enterprise system.
Included in this slice
• frontend-web
• capability-knowledge
• function-embedding
• function-web-search
• function-prompt-engine
• llm-gateway + llm-provider-gpt
• storage-vector
• orchestration-engine + task-router
• YAML configuration files for LLM, routing, capability, and functions
More documentation will be added as the implementation grows.
